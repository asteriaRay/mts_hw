# mts_hw
Учебные проекты ШАД МТС

Небольшой набор учебных задач по статистике и анализу логов (Python + pandas/scipy).

## Содержание
1) Poisson(λ): оценивание параметра и доверительные интервалы  
   `notebooks/hw_5.ipynb`

2) MovieLens: подготовка данных логов и расчет метрик  
   `notebooks/hw6.ipynb`

3) Two-sample hypotheses: сравнение распределений/медиан, бутстрап-подход (A/B-логика)  
   `notebooks/hw7.ipynb`

4) Drift detection: проверка изменений распределений недельных метрик на логах (аналог мониторинга/аномалий)  
   `notebooks/hw8.ipynb`

## Стек
Python, numpy, pandas, scipy, matplotlib.

## Запуск
Открыть ноутбуки в Jupyter/Colab. Данные: MovieLens (GroupLens) — ссылки/загрузка указаны в ноутбуках
